# Reporte de Respuesta ante Incidentes (Live Case)

## Resumen del Proyecto
Este repositorio contiene el informe detallado de un incidente de seguridad en tiempo real detectado en un servidor de producción. Se documenta desde la identificación de la amenaza hasta las propuestas de endurecimiento (hardening).

## Escenario del Ataque
- **Vector de entrada:** Explotación de servicios obsoletos.
- **Malware:** Script bash exfiltrador ejecutado mediante persistencia en Cron.
- **Exfiltración:** Datos sensibles enviados mediante HTTP POST a IP externa.
